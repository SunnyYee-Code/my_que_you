## 3. 任务拆解明细

### T1 账户注册登录与新手引导

**目标**：让新用户能够完成注册、登录并进入产品主流程。

**子任务拆解：**
1. 登录页搭建与登录方式兼容
   - 支持邮箱/手机号/用户名 + 密码登录；
   - 支持验证码校验注册流程。
2. 认证态管理
   - 维护 session、用户信息、角色信息；
   - 支持登录态刷新与退出登录。
3. 新用户 onboarding
   - 首次登录后引导上传头像；
   - 设置用户名并校验规则；
   - 完成后进入社区页。
4. 个人资料基础能力
   - 个人主页展示；
   - 资料编辑、头像更新、昵称修改。

**交付结果：**
- `Login.tsx`
- `AuthContext.tsx`
- `Onboarding.tsx`
- `Profile.tsx`
- `ProfileEdit.tsx`
- `Settings.tsx`

**状态**：✅ 已完成

---

### T2 城市定位与拼团发现

**目标**：让用户进入产品后能快速发现当前城市内可参与的拼团。

**子任务拆解：**
1. 城市上下文能力建设
   - 读取城市列表；
   - 支持本地缓存当前城市；
   - 支持手动切换城市。
2. 地理位置能力接入
   - 获取当前位置；
   - 计算与拼团地点的距离。
3. 社区列表页建设
   - 展示拼团列表；
   - 展示距离、状态、缺口、玩法等信息；
   - 提供状态筛选、距离筛选。
4. 首页引流与附近对局推荐
   - 首页展示附近对局卡片；
   - 支持跳转社区和建局入口。

**交付结果：**
- `CityContext.tsx`
- `useGeolocation.ts`
- `Index.tsx`
- `Home.tsx`

**状态**：✅ 已完成

---

### T3 拼团创建与房主管理

**目标**：让房主可以发起拼团并完成基本管理。

**子任务拆解：**
1. 创建拼团表单
   - 支持时间、地点、人数、玩法、备注等字段；
   - 支持地图选点。
2. 发团前校验
   - 校验用户登录状态；
   - 校验是否已有未结束拼团；
   - 校验每日创建限制。
3. 拼团详情页管理能力
   - 房主查看拼团详情；
   - 取消拼团；
   - 移出成员。
4. 我的拼团页
   - 按“我发起的/我参加的/被移出”分类管理。

**交付结果：**
- `GroupCreate.tsx`
- `GroupDetail.tsx`
- `MyGroups.tsx`
- `AmapLocationPicker.tsx`

**状态**：✅ 已完成

---

### T4 入局申请与审核流转

**目标**：完成用户申请加入拼团到被房主审核通过的闭环。

**子任务拆解：**
1. 申请加入能力
   - 用户从详情页发起申请；
   - 防止重复申请、重复入团。
2. 加入资格校验
   - 校验是否本人为房主；
   - 校验是否已在其它未结束拼团中；
   - 校验是否已有待审核申请。
3. 房主审核页
   - 展示申请列表；
   - 支持同意/拒绝。
4. 审核通过后的状态流转
   - 写入成员数据；
   - 更新申请状态；
   - 同步通知相关方。
5. 成员退出与房主踢人
   - 支持主动退出；
   - 支持房主移出成员并记录理由。

**交付结果：**
- `useJoinGroup.ts`
- `HostRequests.tsx`
- `GroupDetail.tsx`
- `useGroups.ts`

**状态**：✅ 已完成
