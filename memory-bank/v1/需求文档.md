# 雀友聚 V1 需求文档

## 1. 文档目标

本文件用于沉淀 `my_que_you` 在 V1 阶段**已经实现并可被代码验证**的需求范围，仅保留已落地能力，不记录未实现或仅规划中的内容，确保 V1 文档与当前项目真实交付范围一致。

---

## 2. V1 产品目标

V1 的目标是完成同城麻将约局的最小可用业务闭环，让用户可以完成：

1. 注册登录并完成基础资料设置；
2. 进入指定城市浏览附近拼团；
3. 发起拼团或申请加入拼团；
4. 房主审核成员申请并管理拼团；
5. 成员通过群聊、私信、通知完成沟通；
6. 对局后通过评价、信用、举报形成基本治理闭环。

## 5. V1 核心业务闭环

### 5.1 用户找局闭环

1. 用户注册/登录；
2. 首次进入完成新手引导；
3. 选择城市并授权定位；
4. 浏览附近拼团列表并筛选；
5. 查看拼团详情；
6. 提交入局申请；
7. 房主审核通过后加入拼团；
8. 进入群聊沟通并按时到场；
9. 对局结束后完成评价。

### 5.2 房主开局闭环

1. 房主创建拼团；
2. 系统校验是否允许开团；
3. 拼团展示到社区列表；
4. 房主接收申请通知；
5. 房主审核成员；
6. 成局后成员进入聊天；
7. 如有异常可取消拼团或移出成员；
8. 对局结束后沉淀评价与信用。
