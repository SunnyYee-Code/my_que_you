# 雀友聚 V1 测试用例

## 1. 说明

本文档基于 `memory-bank/v1/验收文档.md` 输出可执行测试用例设计，覆盖 V1 需求文档声明的全部场景与功能。测试用例按测试层级拆分为：

- 单元测试（Unit）
- 集成测试（Integration）
- 端到端测试（E2E）
- 补充人工探索测试（Manual）

说明：当前仓库尚未建立完整自动化用例体系，因此本文档先给出建议用例设计与覆盖范围，供后续在 Vitest / Playwright 中逐步落地。

为便于快速查看，本文统一使用 **✅ 已完成** 与 **❌ 未完成** 作为状态标记。
