# 雀友聚 真实环境验收 Checklist

## 文档说明

本文档用于在真实 Supabase 环境中进行端到端验收，确保系统在生产环境下稳定运行。

**验收环境要求**：
- 真实 Supabase 项目（非本地模拟）
- 真实的 Edge Functions 部署
- 真实的 cron 调度配置
- 多个测试账号（普通用户、管理员、超管）
- 真实的短信/推送渠道（如适用）
